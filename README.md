# Static Routing using Cisco Packet Tracer

## Project Overview
This project demonstrates communication between two different networks using static routing in Cisco Packet Tracer.

## Network Topology
- Network 1: 192.168.1.0/24
- Gateway: 192.168.1.1
- Network 2: 192.168.2.0/24
- Gateway: 192.168.2.1
- Router Link: 10.0.0.0/24
- Router 0: 10.0.0.1
- Router 1: 10.0.0.2

## Tools Used
- Cisco Packet Tracer

## Configuration
- Configured router interfaces
- Assigned IP addresses to PCs
- Enabled interfaces using `no shutdown`
- Configured static routes on both routers

## Verification
- Ping Test
- Traceroute Test

## Result
Successfully established communication between devices located on different networks using static routing.
